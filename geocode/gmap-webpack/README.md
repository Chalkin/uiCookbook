# gmap-webpack

> Browserify version converted for the Webpack module bundler, [more info](http://jimfrenette.com/2017/03/google-maps-api-with-webpack/ "Blog post").

```
$ npm install

# develop with webpack-dev-server
$ npm start

# build for production
$ npm run build
```
